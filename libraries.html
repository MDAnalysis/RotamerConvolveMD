Rotamer libraries¶

This package contains the MTSL rotamer library R1A_298K (2011) [Polyhach2011] and an updated MTSL rotamer library R1A_298K (2015) (which is the default library for MTSL in MMM since version 2015.1).

Both libraries were provided by Gunnar Jeschke. The libraries are published under the GPL with his permission.
